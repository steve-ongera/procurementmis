Goods Receipt

How to Receive Goods

  1. Navigate to Goods Receipt > Pending Deliveries
  2. Locate the Purchase Order to receive
  3. Click Receive Goods button
  4. Fill in delivery details (delivery note, date)
  5. Enter quantities delivered for each item
  6. Specify accepted and rejected quantities
  7. Click Create GRN

How to Inspect Goods

  1. Go to Goods Receipt > Receipt History
  2. Click on a GRN to view details
  3. Review delivered items
  4. Select inspection status (Accepted/Rejected/Partial)
  5. Add inspection notes and reasons if applicable
  6. Click Complete Inspection

Handling Quality Issues

If goods do not meet quality standards:

  • Mark items as Rejected or Damaged during GRN creation
  • Provide detailed rejection reasons
  • View all quality issues in Quality Issues page
  • Coordinate with procurement for supplier returns

Stock Management

Adding Stock Items

  1. Go to Inventory > All Stock
  2. Click Add Stock Item
  3. Select store and item
  4. Enter initial quantity and cost
  5. Set reorder level and max stock level
  6. Click Add Stock Item

Monitoring Stock Levels

  • Low Stock Alert: Shows items at or below reorder level
  • Out of Stock: Lists items with zero quantity
  • System automatically flags items needing restock
  • Create requisitions directly from alerts

Stock Takes (Physical Verification)

  1. Navigate to Inventory > Stock Takes
  2. Select the store to verify
  3. Perform physical count of all items
  4. Enter actual quantities in the system
  5. System will show variances
  6. Add remarks for discrepancies
  7. Click Complete Stock Take
  8. System automatically adjusts stock levels

Stock Issues

Creating Stock Issue Requests

  1. Go to Issues > Create Issue
  2. Select store and requesting department
  3. Specify purpose of issue
  4. Add items and quantities needed
  5. Click Create Issue Request

Processing Stock Issues

  1. View pending issues in Issues > Pending
  2. Click Process on an issue
  3. Verify requested quantities against available stock
  4. Adjust quantities if needed
  5. Click Issue Stock
  6. System updates stock levels automatically

Stock Returns

  1. Go to Issues > Returns
  2. Select the original issue to return from
  3. Enter quantities being returned
  4. Provide reason for return
  5. Click Process Return

Stock Transfers

Transferring Stock Between Stores

  1. Navigate to Inventory > Stock Transfers
  2. Select source store (From Store)
  3. Select destination store (To Store)
  4. Choose the stock item to transfer
  5. Enter quantity (system shows available quantity)
  6. Add transfer reason
  7. Click Transfer Stock

Note: System automatically creates movement records for both stores.

Asset Management

Registering Assets

  1. Go to Assets > Register Asset
  2. Enter asset number and tag
  3. Provide description and details (serial, model, brand)
  4. Enter acquisition date and cost
  5. Set depreciation rate and useful life
  6. Assign to department and custodian
  7. Specify physical location
  8. Add warranty information
  9. Click Register Asset

Asset Maintenance Tracking

  • View assets requiring attention in Assets > Maintenance
  • System alerts for warranty expiry
  • Update asset status (Under Maintenance, Damaged, etc.)
  • Schedule and track maintenance activities

Asset Disposal

  1. Go to Assets > Disposal
  2. View assets eligible for disposal (Damaged, Idle)
  3. Click Dispose on selected asset
  4. Enter disposal date and method
  5. Specify disposal value if applicable
  6. Add detailed notes
  7. Click Confirm Disposal

Reports

Available Reports

  • Stock Summary: Overview of stock by store and category
  • Low Stock Report: Items requiring restock
  • Stock Valuation: Total value of inventory
  • Movement Reports: All stock movements by date range

Generating Reports

  1. Navigate to Reports section
  2. Select report type
  3. Apply filters (date range, store, category, etc.)
  4. Click Generate Report
  5. Use Print or Export to Excel buttons
